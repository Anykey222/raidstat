# Безопасность и Отказ от ответственности
Этот проект предоставляется "КАК ЕСТЬ", без каких-либо гарантий, явных или подразумеваемых.

## Использование EXE-файлов
Автор предоставляет скомпилированные версии (.exe) для удобства пользователей. Эти файлы собираются автоматически из исходного кода с помощью GitHub Actions.
Если вы не доверяете исполняемым файлам, вы всегда можете (и должны) проверить исходный код и собрать программу самостоятельно, следуя инструкции в README.md.

## Сообщение о проблемах
Если вы нашли критическую уязвимость или баг, вы можете сообщить об этом автору через личные сообщения.
Однако, автор **не гарантирует** исправление ошибок или выпуск обновлений. Проект разрабатывается в свободное время.

## Конфиденциальность
Программа работает полностью локально.
- Никакие данные (скриншоты, статистика, ники) не отправляются в интернет, кроме случая если вы включили "смешанный" режим OCR - в этом случае скриншоты отправляются в API ocr.space для наилучшего качества распознавания.
- Программа не требует прав администратора.
- Программа не вносит изменений в файлы игры.
